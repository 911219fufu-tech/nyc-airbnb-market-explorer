# Data Directory Guide

## Folder Layout

- `raw/`
  Original source files. Do not edit these in place.
- `processed/`
  Derived files prepared for the website and later analysis.

## Raw Files

### `raw/listings_monthly.csv`
Main monthly panel dataset used for the website.

### `raw/listings.parquet`
Parquet version of listing data.

### `raw/past_rates.csv`
Supplementary rate history dataset.

### `raw/past_rates.parquet`
Parquet version of supplementary rate history.

## Processed Files

These files are generated by:

```bash
python3 scripts/prepare_datasets.py
```

### `processed/listings_monthly_clean.csv`
Cleaned listing-month level dataset for backend analysis and filtered views.

Key changes:

- parses `month_date`
- normalizes `neighborhood` into `zip_code`
- removes rows where `zip_code` cannot be resolved
- normalizes boolean fields
- keeps only the fields needed for the project

### `processed/monthly_overview.csv`
Overall monthly summary across the dataset.

Useful for:

- market-level trend charts
- KPI baselines

### `processed/zip_monthly_summary.csv`
Monthly summary aggregated by ZIP code.

Useful for:

- area comparison
- spatial summaries
- area-level time trends

### `processed/room_type_monthly_summary.csv`
Monthly summary aggregated by room type.

Useful for:

- categorical comparison charts

### `processed/property_type_monthly_summary.csv`
Monthly summary aggregated by property type.

Useful for:

- category comparison and filtering

### `processed/listing_latest_snapshot.csv`
One latest available record per listing.

Useful for:

- default map state
- listing-level snapshots

### `processed/metadata_summary.json`
Small metadata file with date range, filter options, dataset size, and missing-value summary.

## Notes

- `zip_code` in processed outputs is derived from the raw `neighborhood` field.
- Some metrics remain missing after cleaning because they are genuinely absent in the source data.
- The processed outputs are designed for reproducible website development, not as replacements for raw data.
